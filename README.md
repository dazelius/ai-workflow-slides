# AI로 내 업무 자동화하기 — 발표 슬라이드

게임 기획자를 위한 생산성 자동화 가이드 발표용 슬라이드 덱입니다. 브라우저에서 바로 보고, 발표하고, PPT처럼 직접 편집할 수 있습니다.

## 보기 / 발표하기

`slides/index.html`을 열면 슬라이드 목록과 뷰어가 뜹니다.

- `←` `→` 방향키 또는 좌측 목록 클릭으로 슬라이드 이동
- 우측 상단 `⛶` 버튼 또는 `F5`로 전체화면 슬라이드쇼 모드

GitHub Pages로 배포된 경우 별도 서버 없이 이 "보기/발표" 기능은 그대로 동작합니다. (아래 "편집·AI" 기능은 로컬 서버가 있어야 동작합니다.)

## 편집하기 (로컬 서버 필요)

슬라이드 내용 수정, 슬라이드/섹션 추가·삭제·순서변경, AI로 슬라이드 작성/이미지 생성 등은 로컬에서 `server.js`를 켜야 동작합니다 (저장은 실제 파일에 직접 쓰기 때문에, 그리고 AI 기능은 API 키를 서버에서만 다뤄야 하기 때문입니다).

1. `.env.example`을 복사해 `.env`로 만들고, 본인의 API 키를 채워 넣습니다.

   ```
   CLAUDE_API_KEY=...
   OPENAI_API_KEY=...
   ```

2. `slides/start-editor.bat`을 실행합니다 (또는 `slides` 폴더에서 `node server.js`).
3. 브라우저에서 `http://localhost:5500` 접속 → 우측 상단 "편집 모드 켜기".

**`.env`는 절대 커밋/공유하지 마세요.** 이미 `.gitignore`에 포함되어 있어서 `git add`로도 올라가지 않습니다. API 키는 이 로컬 서버 프로세스에서만 사용되고 브라우저(클라이언트)로는 전달되지 않습니다.

## 폴더 구조

```
slides/
  index.html      슬라이드 목록 + 뷰어 (덱 뷰어)
  server.js       로컬 편집/저장/AI API 서버 (Node 기본 모듈만 사용)
  assets/         공용 스타일·에디터 스크립트·슬라이드 목록(deck.json)
  deck/           슬라이드 개별 HTML 파일
```

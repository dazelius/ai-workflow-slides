ERD 자동화

구현 방식

1
엑셀에서 Define 추출
각 테이블 Define이 담긴 Excel 파일에서 스키마 정보를 파싱
2
Python 스크립트 변환
추출한 데이터를 DBML(Database Markup Language) 형식으로 텍스트 치환
3
dbdiagram.io 렌더링
생성된 DBML을 웹 서비스에 입력하면 자동으로 ERD 시각화
핵심 이점
  • 엑셀 → DBML 자동 변환으로 수작업 제거
  • 스키마 변경 시 스크립트 재실행만으로 ERD 갱신
  • 팀원 누구나 같은 방식으로 ERD 관리 가능
  • 버전 관리 시스템에 DBML 파일 보관 가능
